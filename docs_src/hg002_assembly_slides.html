Zygosity annotation

  • Heterozygous — one ref + one alt allele; carrier state
  • Homozygous — both alleles are the same alt; full penetrance expected
Homozygous pathogenic variants are prioritised — both alleles carry the disease-associated change, so phenotypic effect is most likely.
Pathogenic table in the report includes coordinate, REF→ALT, gene, zygosity, disease name, and ClinVar review status.
